# Campos do formulário de cadastro  

### Supondo que o formulário seja para um cadastro "básico" de uma pessoa  

## Info pessoal  
- Nome  
- Sobrenome  
- Data de nascimento    
- Gênero  

## Endereço  
- Rua  
- Bairro  
- Nº  
- Complemento  
- CEP  

## Info adicional  
- Campo observações  

## Usuário e Senha  
- Nome de usuário  
- Crie uma senha  
- Repita a senha  

## Botão "Finalizar cadastro"  
- Botão para finalizar cadastro
